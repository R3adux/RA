ReaduxAdditions (RA)

A collection of small tweaks and quality-of-life additions for KSP RO/RP1.
Included Mods:

    KCT-Resizer: Adjusts the GUI heights of Kerbal Construction Time 
    (Integration and Combined lists) for better visibility.

Installation:

    Download the latest .zip from the Releases section (or SpaceDock).

    Extract the ReaduxAddition folder into your KSP GameData directory.

Dependencies:

    none (just RP-1)
